### <a name="using-a-data-section"></a><span data-ttu-id="65d75-140">使用数据部分</span><span class="sxs-lookup"><span data-stu-id="65d75-140">Using a Data Section</span></span>

<span data-ttu-id="65d75-141">若要使用 Data 节的内容，请将其分配给变量并使用变量表示法来访问内容。</span><span class="sxs-lookup"><span data-stu-id="65d75-141">To use the content of a Data section, assign it to a variable and use variable notation to access the content.</span></span>

<span data-ttu-id="65d75-142">例如，以下 data 节包含一个 `ConvertFrom-StringData` 命令，该命令将此处的字符串转换为哈希表。</span><span class="sxs-lookup"><span data-stu-id="65d75-142">For example, the following data section contains a `ConvertFrom-StringData` command that converts the here-string into a hash table.</span></span> <span data-ttu-id="65d75-143">将哈希表分配给该 `$TextMsgs` 变量。</span><span class="sxs-lookup"><span data-stu-id="65d75-143">The hash table is assigned to the `$TextMsgs` variable.</span></span>

<span data-ttu-id="65d75-144">`$TextMsgs`变量不是 data 节的一部分。</span><span class="sxs-lookup"><span data-stu-id="65d75-144">The `$TextMsgs` variable is not part of the data section.</span></span>

```powershell

    ConvertFrom-StringData -StringData @'

Text002 = Windows Server 2008 R2
'@
}
```

<span data-ttu-id="65d75-145">若要在中访问哈希表中的键和值 `$TextMsgs` ，请使用以下命令。</span><span class="sxs-lookup"><span data-stu-id="65d75-145">To access the keys and values in hash table in `$TextMsgs`, use the following commands.</span></span>


```powershell

'@

$TextMsgs
```

<span data-ttu-id="65d75-148">结果与前面的示例相同。</span><span class="sxs-lookup"><span data-stu-id="65d75-148">The result is the same as the previous example.</span></span>
