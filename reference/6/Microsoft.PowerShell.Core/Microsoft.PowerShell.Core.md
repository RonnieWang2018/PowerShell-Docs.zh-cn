# Microsoft.PowerShell.Core 模块

## 说明

本部分包含随 PowerShell Microsoft Core 模块一起安装的 cmdlet 的帮助主题。 核心模块包含用于管理 PowerShell 的基本功能的 cmdlet 和提供程序。

## Microsoft PowerShell Cmdlet

### [Add-History](Add-History.md)
向会话历史记录附加条目。

### [Clear-History](Clear-History.md)
从 PowerShell 会话命令历史记录中删除条目。

### [Clear-Host](Clear-Host.md)
清除主机程序中的显示内容。

### [Connect-PSSession](Connect-PSSession.md)
重新连接到已断开连接的会话。

### [Debug-Job](Debug-Job.md)
调试正在运行的后台工作流作业。

### [Disable-ExperimentalFeature](Disable-ExperimentalFeature.md)
在启动 PowerShell 的新实例时禁用实验功能。

### [Disable-PSRemoting](Disable-PSRemoting.md)
阻止 PowerShell 终结点接收远程连接。

### [Disable-PSSessionConfiguration](Disable-PSSessionConfiguration.md)
禁用本地计算机上的会话配置。

### [Disconnect-PSSession](Disconnect-PSSession.md)
断开与会话的连接。

### [Enable-ExperimentalFeature](Enable-ExperimentalFeature.md)
启动新的 PowerShell 实例时启用实验性功能。

### [Enable-PSRemoting](Enable-PSRemoting.md)
将计算机配置为接收远程命令。

### [Enable-PSSessionConfiguration](Enable-PSSessionConfiguration.md)
启用本地计算机上的会话配置。

### [Enter-PSHostProcess](Enter-PSHostProcess.md)
连接到并进入与本地进程的交互会话。

### [Enter-PSSession](Enter-PSSession.md)
启动与远程计算机的交互式会话。

### [Exit-PSHostProcess](Exit-PSHostProcess.md)
关闭包含本地进程的交互式会话。

### [Exit-PSSession](Exit-PSSession.md)
结束与远程计算机的交互式会话。

### [Export-ModuleMember](Export-ModuleMember.md)
指定要导出的模块成员。

### [ForEach-Object](ForEach-Object.md)

### [Get-Command](Get-Command.md)

### [Get-ExperimentalFeature](Get-ExperimentalFeature.md)

### [Get-Help](Get-Help.md)

### [Get-History](Get-History.md)
获取在当前会话过程中输入的命令列表。

### [Get-Job](Get-Job.md)
获取当前会话中正在运行的 PowerShell 后台作业。

### [Get-Module](Get-Module.md)
获取已导入或可导入到当前会话的模块。

### [Get-PSHostProcessInfo](Get-PSHostProcessInfo.md)
获取有关 PowerShell 主机的进程信息。

### [Get-PSSession](Get-PSSession.md)

### [Get-PSSessionCapability](Get-PSSessionCapability.md)
获取受约束的会话配置上的特定用户的功能。

### [Get-PSSessionConfiguration](Get-PSSessionConfiguration.md)
获取计算机上已注册的会话配置。

### [Import-Module](Import-Module.md)
将模块添加到当前会话。

### [Invoke-Command](Invoke-Command.md)

### [Invoke-History](Invoke-History.md)

### [New-PSTransportOption](New-PSTransportOption.md)

### [Out-Host](Out-Host.md)

### [Out-Null](Out-Null.md)

### [Save-Help](Save-Help.md)

### [Set-PSDebug](Set-PSDebug.md)

### [Set-PSSessionConfiguration](Set-PSSessionConfiguration.md)

### [Unregister-PSSessionConfiguration](Unregister-PSSessionConfiguration.md)

### [Update-Help](Update-Help.md)

### [Wait-Job](Wait-Job.md)
